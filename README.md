# ecgsyn_wr (WIP)
A desktop application to design Wistar Rat (WR) beats from ECG parameters.

## Installation

```
pip install git+https://github.com/sfcaracciolo/ecgsyn_wr.git
```

## Usage
```
from ecgsyn_wr import App
App.run()
```
